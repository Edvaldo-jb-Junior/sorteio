# 🎲 Sorteio

Uma aplicação web desenvolvida para realizar sorteios de forma rápida, simples e intuitiva. O projeto permite adicionar participantes e selecionar um vencedor aleatoriamente,
proporcionando uma experiência prática para eventos, promoções, dinâmicas e atividades em grupo.

## 📖 Sobre o Projeto

O Sorteio foi desenvolvido com o objetivo de praticar conceitos de desenvolvimento front-end, manipulação do DOM e lógica de programação utilizando JavaScript.

A aplicação oferece uma interface amigável onde os usuários podem cadastrar participantes e executar sorteios instantaneamente.

## ✨ Funcionalidades

- ➕ Adicionar participantes
- 📝 Listar participantes cadastrados
- 🎲 Realizar sorteio aleatório
- 🏆 Exibir o vencedor
- 🔄 Reiniciar sorteio
- 📱 Layout responsivo
- ⚡ Interface simples e intuitiva
